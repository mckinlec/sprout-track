### Previous Auth Issue (Resolved)

The 401 error was caused by a **truncated device token** (50 chars instead of the required 64). The token `4768640b...` from the original handoff was incomplete. The correct full-length token resolves the issue — no code bug existed.

Device tokens are 64-char hex strings generated by `randomBytes(32).toString('hex')`. Always copy the full token from the Settings UI.

## Key Fixes Applied

### Timezone Bug (Today Stats)
The server runs in Docker with UTC timezone. `todayStart.setHours(0,0,0,0)` calculated midnight UTC, causing "today" stats to reset at 7 PM Eastern. Fixed by adding `getMidnightEastern()` to both `/api/ha/query` and `/api/ha/status` — uses `Intl.DateTimeFormat` with `timeZone: 'America/New_York'` and handles EDT/EST offset.

### Pump Support
Added `pump-start` and `pump-end` actions to `/api/voice/log`. HA scripts start a 15-minute `timer.pump_session` and an automation fires a TTS notification via Piper when the timer finishes.
